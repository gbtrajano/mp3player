# Meu player de MP3 em PYTHON - Gabriel Trajano

## Esse código é capaz de reproduzir arquivos MP3 através de código.

### Como instalar meu projeto

Primeiro copie a URL do projeto indo em "<>Código", na aba HTTPS copie a URL.

```bash
https://github.com/gbtrajano/mp3player.git
```

E em seguida em seu terminal utilize o seguinte comando:

```bash
git clone https://github.com/gbtrajano/mp3player.git
```

### Instalação de Dependências

```bash
pip install -r requirements.txt
```

### Bibliotecas Requiridas

- pygame
